
# Cursor 使用教程

Cursor 是一款革命性的代码编辑器，它将传统的编辑功能与先进的 AI 技术完美结合，为开发者提供了一个智能、高效的编程环境。本文档将帮助你快速上手和深入了解 Cursor。
